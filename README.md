# LifeStage
Welcome to the LifeStage project! This repository contains the code for a web application designed to help users track and manage different stages of their lives. The application is built using Angular for the frontend and Spring Boot for the backend API to handle data storage and retrieval.

## Running the application locally
To run the frontend part of this application locally, see the guide in [frontend/lifestage/FRONTEND_README.md](frontend/lifestage/FRONTEND_README.md).

To run the backend part of this application locally, see the guide in [backend/lifestage/BACKEND_README.MD](backend/lifestage/BACKEND_README.MD).
